Pengantar

HTML (Hyper Text Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web. HTML menjadi dasar dari sebuah halaman website. sebuah file HTML di di simpan dengan ekstensi .html Untuk membuat sebuah website tidak cukup hanya menggunakan HTML, diperlukan bantuan CSS, JavaScript dan PHP untuk membuat website yang dinamis. Halaman website yang dibuat hanya menggunakan HTML maka halaman website tersebut di sebut halaman statis karena tidak memiliki aksi atau fungsi-fungsi yang dapat mengelola website.

Fitur Utama

  • Membuat Tabel
  • Membuat Daftar
  • Membuat Link: Contoh Link ke akun GitHub

Urutan Langkah Belajar HTML

  3. Berlatih membuat program HTML
